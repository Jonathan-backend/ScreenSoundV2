# 🚧 Status do Projeto

> **Em desenvolvimento**

Este repositório reúne exemplos práticos e experimentos desenvolvidos durante meus estudos de conceitos fundamentais da plataforma .NET, com foco em programação genérica, proxies e acesso a banco de dados utilizando ADO.NET.

---

## 📖 Sobre o projeto

O objetivo deste projeto é consolidar conhecimentos sobre recursos essenciais do ecossistema .NET por meio da implementação de exemplos práticos.

Os estudos abrangem desde o uso de **Genéricos (Generics)** para criar código reutilizável e tipado, até conceitos de **Proxies** e acesso direto a banco de dados com **ADO.NET**.

---

## 🎯 Objetivos

- Compreender o funcionamento de Generics
- Criar classes e métodos genéricos
- Estudar Proxies e sua utilização
- Aprender acesso a banco de dados utilizando ADO.NET
- Manipular conexões e comandos SQL
- Aplicar boas práticas de desenvolvimento em C#
